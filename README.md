# AtCoder

AtCoderのコンテスト問題を難度別に管理するリポジトリです。

## ディレクトリ構造

```bash
.
├── a/                    # A問題（序問題）
│   ├── 001-100/
│   ├── 301-400/
│   └── 401-500/
├── b/                    # B問題（第2問）
│   ├── 301-400/
│   └── 401-500/
├── c/                    # C問題（第3問）
│   ├── 301-400/
│   └── 401-500/
└── README.md
```

## 言語

- C++ (.cpp)
- C (.c)

## コンテスト範囲

- **001-100**: ABC001～ABC100
- **301-400**: ABC301～ABC400
- **401-500**: ABC401～ABC500
