# AtCoder

AtCoderのコンテスト問題を難度別に管理するリポジトリです。

## 参加コンテスト

### ABC（AtCoder Beginner Contest）

AtCoderの初級者〜中級者向けコンテスト。A〜G問題で構成。

- 参加回: ABC001〜、ABC411〜 を中心に参加
- 取り組み問題: A問題（全回）、B・C問題（一部）、D問題（一部）

### AWC（AtCoder Weekly Contest）

毎週開催されるコンテスト。A〜D問題で構成。

- 参加回: AWC0002, AWC0003, AWC0005, AWC0006, AWC0007
